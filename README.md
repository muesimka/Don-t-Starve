## 📋 О проекте

**Don't Starve Clone** — это браузерная игра в жанре survival, разработанная на чистом JavaScript с использованием Canvas API. Проект находится в активной фазе разработки и поддержки.

---

## 🎯 Mission Objective

> [!IMPORTANT]
> **Команда, внимание!**

Проект входит в фазу активной поддержки и полировки. Теперь каждый из вас несёт ответственность за качество продукта. Вы — команда поддержки!

### 📌 Что это значит?

| Задача | Описание |
|--------|----------|
| 🔍 **Поиск багов** | Самостоятельно находите проблемы в коде |
| 🔧 **Исправление** | Устраняете найденные неполадки |
| 💎 **Оптимизация** | Чистите и улучшаете существующий код |
| ✨ **Улучшения** | Добавляете новый функционал без напоминаний |
| 📝 **Документация** | Фиксируете все изменения |

---

## ✨ Улучшения кода: Что разрешено

### ✅ Приветствуется

- Чистка неиспользуемого кода
- Рефакторинг дублирующихся участков
- Оптимизация производительности
- Улучшение читаемости
- Добавление комментариев
- Удаление лишних `console.log`

### 🚀 Самостоятельная модификация

Вы можете добавлять:

- Новые типы врагов
- Новые ресурсы и предметы
- Новые рецепты крафта
- Звуковые эффекты
- Визуальные улучшения

> [!TIP]
> **Любая самостоятельная доработка засчитывается как домашняя работа!**
